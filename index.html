CSS

  • There are three ways to style a webpage using CSS
    1. Inline CSS
    2. Internal CSS Style Sheet
    3. External Style Sheet
  • Inline is used to change one single attribute
  • Internal Css style sheet is used to style the document in the header section.
  • External style sheet is the most commonly used way to style a webpage. This keeps all CSS rules in one file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The external CSS style sheet is linked in the headder of the HTML page.
